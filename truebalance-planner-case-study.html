TrueBalance Planner

Personal Debt Elimination & Financial Planning Platform

TrueBalance Planner is a web-based financial planning application designed to help users organize debt obligations, evaluate payoff strategies, visualize progress, and create realistic paths toward financial freedom.

Product Overview

Problem

Many consumers struggle to visualize debt payoff timelines, prioritize balances, and understand the long-term impact of interest rates.

Goal

Create an easy-to-use planning tool that transforms debt data into actionable payoff strategies.

Success Criteria

  • Clear payoff visibility
  • Multiple payoff strategies
  • Mobile-friendly design
  • Data privacy
  • Simple onboarding

Requirements Analysis

Requirement ID Requirement Business Value
FR-001 Add debt accounts Allows users to centralize debt obligations.
FR-002 Calculate payoff timeline Helps users understand payoff duration.
FR-003 Compare payoff strategies Supports Snowball and Avalanche decision-making.
FR-004 Track balances and payments Shows progress over time.
FR-005 Generate charts Makes debt reduction visually understandable.
FR-006 Import CSV data Supports faster setup and repeatable data loading.
FR-007 Export or backup data Supports user control and portability.

Lessons Learned

Designing financial tools requires balancing analytical accuracy with usability. This project reinforced the importance of requirements gathering, iterative testing, user-centered design, data visualization, and clear decision-support workflows when creating tools that help users make financial decisions.
